FAQ

Quick answers and links.

Where do I start?
Start with the Main Macro Installer, then pick the macros you want from the Projects section.
Where do I report issues?
Head to the Discord server — that’s where most bug reports and support requests go first. If it’s repo-specific you can still open a GitHub issue after chatting there.
Can I see all projects at once?
Yes — visit All repositories.
Do the macros auto-update?
The installer checks each repo for new releases when you launch it. If a macro has an update, you’ll see the release notes and can apply it with one click.
Can I customize the workflows?
Use Macro Creator to tweak timings and sequences. It exports configs that the installer can load, so you don’t have to edit scripts manually.
What if I’m running offline?
The installer ships with the latest versions baked in, so everything works offline. Once you reconnect, it will sync any pending releases.
